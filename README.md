# @amlplugins/deepgram-agent

Native `@deepgram/sdk` SDK re-exported under the `@amlplugins` namespace.

> Deepgram Voice Agent — full-duplex Listen+LLM+Speak pipeline over WebSocket, tools, barge-in handling.

## Install

```bash
npm install @amlplugins/deepgram-agent
```

## Use

```ts
import { /* named exports from @deepgram/sdk */ } from "@amlplugins/deepgram-agent";
// or
import sdk from "@amlplugins/deepgram-agent";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
